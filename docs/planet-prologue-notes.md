# Planet Prologue — 작업 노트 (임시)

> 빛의 행성(Planet1) Prologue 화면을 기준으로 만들고, **행성 1~4가 재사용**한다.
> 시각 목표: 사용자가 준 기획안 스크린샷(빛의 행성 프롤로그).
> 관련: [scene-development-process.md](scene-development-process.md), [Prologue.tsx](../src/scenes/planet/planet1/Prologue.tsx)

## 전역 원칙 (모든 요소에 적용)

1. **재사용 단일 컴포넌트** — 레이아웃은 한 번만 만들고 행성별 데이터(텍스트/보상/스텝)와
   색상만 주입한다. 데이터의 props/theme화는 9개 요소 모두 확정한 뒤 일반화(아직은 planet1 inline).
2. **색은 CSS, 캐릭터만 이미지** — 색이 안 변하는 고유 그림(하티/마스코트/우주선/보상아이콘)은
   `<img>`(또는 이모지 임시), 패널 배경·테두리·헤더·라벨·버튼색은 CSS로.
3. **행성색 = CSS 변수 한 세트** — `.prologue`에 `--planet-accent` 계열 변수로 받아
   한 번에 갈아끼운다. planet1은 따뜻한 골드/오렌지.
   - `--planet-accent` (헤더바/배너 포인트), `--planet-accent-deep`(테두리/제목텍스트),
     `--planet-panel-bg`(패널 배경), `--planet-tab`(오늘의 탐험 탭 — 2단계).

## 요소별 결정

### ① 타이틀 배너 — ✅ 확정
- 기존 mission1의 `.prog-title` 마크업/스타일 그대로 재사용, 글씨만 "빛의 행성".
- 위치·크기 동일하게(top:34+margin14 ≒ top:48, left:44, 흰 알약형, ⭐ + 텍스트).
- mission.css에 의존하지 않도록 동일 규칙을 Prologue.css에 복제.

### ② 학습 목표 패널 — ✅ 확정
- 스크린샷과 유사하게 CSS로. 🎯"학습 목표" 헤더 + ⭐ 불릿 3개(이모지 사용).
- 색은 Claude가 비슷하게(따뜻한 오렌지 헤더 + 크림 배경). 마음에 안 들면 사용자가 색코드 제공.
- 문구: 공감이 왜 필요한지 / 마음을 이해하려 노력하는 방법 / 공감이 사람을 연결·행복하게 함.

### ③ 하티 캐릭터 + 말풍선 — ✅ 확정
- 캐릭터: 이미지 `/assets/char/Hati/hati_robot_worried.png`.
- 말풍선: 흰색 배경(색 고정). 대사 "이해의 빛이 사라졌어… 친구들의 마음을 이해하면 빛을 되찾을 수 있어!"
- "하티" 이름표: 색 고정(하티 브랜드 그린)이지만 CSS로 구현.

### ④ 메인 일러스트 — ✅ 확정
- 현재 배경 이미지(`light-planet-stage1.png`)로 충분. 추가 작업 없음.

### ⑤ 획득 가능한 보상 패널 — ✅ 확정
- ②와 거의 동일한 패널. ✨"획득 가능한 보상" 헤더 + 아이템 3개(아이콘+이름+설명).
- 아이콘은 이모지 임시(💎 이해의 원석 / 🪞 공감 거울 / ❤️ 공감 에너지 경험치). 추후 그림 교체.

### ⑥ "오늘의 탐험" 패널 — ✅ 확정
- 좌하단 패널. 마스코트 이미지 `/assets/char/planet1-lumi.png`(한글명 `행성1-루미.png` → ASCII로 rename).
- 탭 라벨 "오늘의 탐험": **빨강 대신 행성색**(`--planet-accent`).
- 제목 "사라진 이해의 빛을 찾아라!" + 부제 "친구들의 마음을 이해하고 빛을 되찾아보자!".

### ⑦ "탐험 시작!" 버튼 — ✅ 확정
- 기존 `.btn`(전역) 그대로 재사용, ⑥ 패널 안으로 이동.

### ⑧ 미션 흐름(스텝) 인디케이터 — ✅ 확정
- 하단 가로 띠. 번호 배지(행성색) + 라벨 + 화살표.
- 스텝 텍스트는 **행성별 데이터** → 컴포넌트의 `STEPS` 배열. planet1:
  마음 신호 탐색하기 / 얼어붙은 공감 거울 깨우기 / 공감없는 세상으로.

### ⑨ "우주선으로 이동" 버튼 — ✅ 확정
- 우하단, 이미지 `/assets/char/SpaceshipIcon.png` + 텍스트. 동작: `onHome()` → `/home`.
- 컨테이너 `planet1/index.tsx`에서 `onHome={() => nav("/home")}` 주입.
